Detailed Description

libsocket++ is the C++ wrapper around libinetsocket/libunixsocket

This module contains all classes and functions which are part of libsocket++, i.e. all classes and some friend functions. Note: To obtain informations about errors, use errno. You may find possible values in the syscall's man pages.
